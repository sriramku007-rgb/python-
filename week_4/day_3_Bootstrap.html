3. Explain the difference between container and container-fluid in Bootstrap. When would you use each? Fixed-width container Provides a responsive fixed width that changes at different screen sizes (using breakpoints). The width is automatically adjusted based on the screen size, but it never takes the full width of the viewport. Full-width container Always takes 100% width of the screen, regardless of device size. It stretches across the entire viewport.
